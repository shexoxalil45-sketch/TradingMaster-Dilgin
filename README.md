# Dilgin Trader Bot (Dark) - Full package
This repo contains the UI and server for the Dilgin Trader Bot. Run locally on Android (Termux) or Windows.

Quick start (Termux):
1. pkg update && pkg upgrade -y
2. pkg install python git -y
3. git clone https://github.com/<YourUserName>/TradingMaster-Dilgin.git
4. cd TradingMaster-Dilgin
5. pip install --upgrade pip
6. pip install -r requirements.txt
7. python app.py
Open http://127.0.0.1:5000 in your phone browser.
